# Impacto del conflicto Rusia-Ucrania en los precios de los alimentos en Europa (2020-2025)

## Descripción

Proyecto desarrollado para analizar la evolución de los precios de alimentos en Europa utilizando datos del World Food Programme (WFP).

## Objetivo

Analizar posibles efectos del conflicto Rusia-Ucrania sobre los precios alimentarios europeos.

## Dataset

Global Food Prices Database (WFP)

## Tecnologías

- Python 3.12
- Pandas
- NumPy
- Jupyter Notebook
- Git
- GitHub
